# 🟫 MC 大冒险 — 我的世界版马里奥

一个用 HTML5 Canvas 写的横版闯关游戏（纯前端，零依赖）。把马里奥世界换成《我的世界》：主角是 Steve，反派是 MC 怪物和 4 大 Boss，路上全是 MC 机关。

## ▶ 怎么玩

**最简单**：双击 `MC大冒险-单文件版.html`，用浏览器（Chrome / Edge 推荐）打开即可，零安装、零依赖。

> 第一次有声音前需要先按一下键盘 / 点一下屏幕（浏览器策略要求用户交互后才能播放音效）。

也可以双击 `index.html`（外壳 + `game.js` 分离版），或用本地服务器（见下方「开发」）。

## 🎮 操作

| 操作 | 按键 |
|------|------|
| 移动 | `←` `→` 或 `A` `D` |
| 跳跃 / 二段跳 | `空格` / `↑` / `W`（空中再按一次= 二段跳，长按跳更高）|
| 挥剑攻击 | `J` 或 **鼠标左键** |
| 射箭（弓） | `K`（远程攻击，可打怪与 Boss，有冷却）|
| 冲刺 | `Shift`（移动时按，短时间加速）|
| 暂停 / 设置 | `P` 或 `Esc`（菜单内可调音量、静音、画面震动）|
| 快速静音 | `M` |
| 开始 / 重试 | `Enter` |

手机/平板：屏幕会自动出现虚拟按键（含右上角暂停键）。

> 操作手感细节：支持 **coyote time**（离开平台瞬间仍可起跳）和**跳跃缓冲**（落地前提前按也能跳）。

## 💾 存档与设置

历史最高分与所有设置（音量 / 静音 / 画面震动）会自动保存在浏览器本地（localStorage），下次打开自动恢复。失焦或切到后台会**自动暂停**。

## 🪤 MC 机关

- **TNT**：靠近/踩上会点燃倒计时然后爆炸，连锁炸毁周围方块（也能炸怪、炸 Boss）
- **压力板**：踩上去触发两侧暗箭
- **岩浆**：碰到即受重伤
- **尖刺 / 仙人掌**：踩上去受伤

## 💎 收集 & 战斗

- **绿宝石**：加分
- **金苹果**：回血
- 击杀方式：**踩头**（踩在怪物头顶）、**挥剑**或 **射箭**；Boss 同理，三种方式都能造成伤害
- **踩头有奖励**：成功踩踏怪物 / Boss 会获得短暂无敌帧，踩踏本身不掉血
- Boss 顶部有**血条 + 数字**，攻击时可直观看到血量下降
- 4 颗心（8 点生命），受伤有短暂无敌帧

## 🛠 技术说明

- `index.html` — 页面外壳 + 自适应缩放 + 虚拟按键
- `game.js` — 全部游戏逻辑：固定时间步长主循环、瓦片物理引擎、摄像机、怪物/Boss AI、粒子系统、Web Audio 程序化音效、暂停/设置菜单、存档、关卡数据
- 关卡用字符画在 `game.js` 顶部的 `LEVELS` 数组里，想改地图直接编辑那些字符串即可（图例见代码注释）
- **帧率无关**：逻辑固定 60Hz 步进、渲染随刷新率，120/144Hz 显示器上速度一致

## 🧑‍💻 开发 / 构建 / 测试

需要 Node.js 18+。首次拉取后：

```bash
npm install            # 安装开发依赖 (仅 Playwright, 用于自动化测试)

npm start              # 启动本地开发服务器  http://localhost:8080
npm run build          # 内联生成单文件版 (dist/ + 根目录的 单文件版.html)
npm test               # headless Chromium 冒烟测试 + 截图到 tests/screenshots/
npm run check          # 仅做 game.js 语法检查
```

> 改完 `game.js` / `index.html` 后请 `npm run build` 重新生成单文件版，二者保持同步。
> 运行 `npm test` 前需先执行一次 `npx playwright install chromium` 下载浏览器内核。

### 目录结构

```
index.html                  页面外壳 (引用 game.js)
game.js                     游戏全部逻辑
MC大冒险-单文件版.html       构建产物: 双击即玩的单文件
build.mjs                   构建脚本 (内联打包)
server.mjs                  零依赖静态开发服务器
tests/smoke.mjs             headless 冒烟测试
dist/                       构建输出
```

## 📦 交付与部署

- **本地分发**：直接给对方 `MC大冒险-单文件版.html` 一个文件即可。
- **静态托管**（GitHub Pages / Netlify / 任意静态服务器）：上传 `dist/` 内容，入口 `index.html`。
