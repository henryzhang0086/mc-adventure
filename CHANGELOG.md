# 更新日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.6.0] — 2026-06-25

视觉 2.0（Roadmap v1.6）。

### 新增
- **深暗之域光照遮罩**：监守者关卡改为跟随玩家的径向光照，只照亮周围一圈、越远越黑，配合幽蓝暖芯，紧张感拉满。
- **草原昼夜循环**：第一章天空在白天/黄昏/夜晚间缓慢渐变，夜晚浮现星空与月亮、白天为暖阳。
- **关卡淡入过场**：每次进入关卡 / 竞技场 / 新一波，黑场淡入切换更有仪式感。

## [1.5.0] — 2026-06-25

游戏模式与重玩性（Roadmap v1.5）。

### 新增
- **标题模式选择**：冒险 / Boss Rush / 无尽生存，↑↓ 或点击选择。
- **Boss Rush**：在专属竞技场连续挑战 4 大 Boss，全程计时，结算显示用时。
- **无尽生存**：一波波刷怪（波数越高越多、越后期解锁末影人/烈焰人），每 3 波掉落金苹果，看能坚持到第几波；结算显示波数。
- HUD 与结算页按模式显示进度（Boss N/4 + 用时 / 第 N 波）。

## [1.3.0] — 2026-06-25

打击感、视听氛围与手游操作体感升级（Roadmap v1.3）。

### 新增
- **背景音乐**：程序化 16 步音序器，每章一段循环旋律 + 贝斯（草原 / 下界 / 深暗 / 末地），**Boss 战自动切换更激烈的曲子**，结算/标题自动停。走独立 music 增益再进 master，受全局音量/静音控制；Web Audio 提前调度，节拍与帧率无关。
- **打击感（juice）**：命中 / 爆炸 / 受击 **顿帧**（hit-stop，冻结逻辑数帧）、**全屏闪光**（爆炸橙、击败白、受击红）、**伤害飘字**（怪 `-1`、Boss `-N`、得分 `+N`、拾取 `+10/+❤`）。
- **连击系统**：连续击杀累计连击数，屏幕中上显示「连击 ×N」（越高越大越黄），并给额外连击奖励分。
- **摄像机升级**：朝移动方向**前瞻**多看一截 + **平滑跟随**（关卡/Boss 切换首帧吸附不滑入）。
- **氛围天气**：草原飘落花粉、下界上升火星、末地紫色微尘、深暗加密幽蓝孢子，叠加按世界深浅的**暗角**。

### 手游操作体感（按反馈）
- **按住连续攻击**：剑 / 弓改为按住即按冷却自动连发，手机上不用狂点。
- **双击方向冲刺**：移动端无 Shift，双击左 / 右方向即可冲刺（保留键盘 Shift）。
- **震动反馈**：跳跃 / 攻击 / 受击 / 爆炸触发 `navigator.vibrate`（支持的设备）。
- **虚拟按键增强**：按钮加大到 72px、按下高亮 + 回弹动画，反馈更明确。

## [1.2.0] — 2026-06-24

移动端可用性修复。

### 修复
- **横屏无法移动**（重要）：虚拟按键默认 `touch-action:auto`，横屏时按左右键的轻微横向滑动被浏览器当成滚动手势而取消触摸；现给所有触摸控件设 `touch-action:none` 彻底解决。
- **电脑操作提示遮挡画面**：触屏设备（`pointer:coarse`）下隐藏底部键盘提示条。
- **触摸控制重写**：改为全局统一追踪所有触摸点（按坐标命中按钮），正确支持多指同按（如移动+跳跃）、手指滑动切换按钮、`touchcancel`，修复真机多指/滑动时按键丢失或卡住的问题。
- 增加刘海屏安全区适配（`viewport-fit=cover` + `env(safe-area-inset-*)`），横屏时按键不被刘海/home 指示条遮挡。

## [1.1.0] — 2026-06-24

根据玩家反馈新增内容与修复。

### 新增
- **弓箭技能**：`K` 键（手机🏹按钮）射箭，可远程攻击小怪与 Boss，附拉弓姿势与冷却。
- **Boss 血条数字化**：血条新增 `当前/最大` 数字、每点血一格刻度、低血量变色，攻击时一目了然。

### 修复
- **Boss 血条不显示**（重要）：原先血条被画在世界坐标里，Boss 战摄像机锁定后被推出屏幕外，全程看不到血量；现移至 HUD 屏幕层正常显示。
- **踩踏 Boss 会被反伤**：踩头成功现给予短暂无敌帧，踩踏造成伤害但玩家不掉血（踩小怪同理），奖励技术操作。

### 其他
- 新增 `?debug` URL 参数的 QA 调试钩子（`__mc.warpToBoss()` 等），仅调试时启用。

## [1.0.0] — 2026-06-24

把原型打磨为可交付的工业级产品。

### 新增
- **暂停 / 设置菜单**（`P` / `Esc`）：音量调节、静音、画面震动开关，键鼠 + 触屏均可操作。
- **存档持久化**（localStorage）：历史最高分、所有设置自动保存，标题页 / 结算页 / HUD 展示最高分。
- **移动端暂停键**，触屏按钮全部支持鼠标按下（便于桌面调试）。
- 运行时**崩溃保护层**：捕获异常并显示提示，避免黑屏。
- 失焦 / 切后台**自动暂停**；canvas 上下文丢失自动恢复。
- 交付工具链：`npm start`（开发服务器）、`npm run build`（生成单文件版）、`npm test`（headless 冒烟测试 + 截图）。

### 修复
- **高分屏毛刺**：canvas 现按设备像素比（DPR）渲染，实际像素 = 屏幕物理像素（整数倍超采样，上限 6），并改用平滑缩放。4K/5K 上文字、爱心、圆形、渐变不再有锯齿/毛刺；DPR 变化（拖到不同缩放的显示器）会自动重新适配。
- **帧率无关性**：改为固定时间步长（60Hz 逻辑）累加器循环，解决在 120/144Hz 高刷新率显示器上游戏速度翻倍的严重问题。
- 音频在不支持 / 被拦截时不再抛错（统一经主音量节点输出，可全局调音量 / 静音）。
- 单帧多次逻辑步进时，单次按键只生效一次（避免一帧内误触发二段跳等）。
- 死亡延时结算加状态判定，避免与重开竞态。

### 改进
- **操作手感**：新增 coyote time（离开平台后短暂仍可起跳）与跳跃缓冲（落地前提前按也能跳）。
- 缺失 DOM 元素时输入绑定不再报错（健壮性）。
